Text Spacing

Properties of Text Spacing

There are 5 properties to control text spacing in CSS:

  1. text-indent
  2. letter-spacing
  3. line-height
  4. word-spacing
  5. white-space

1. Text-Indent Property

Defines the indentation (spaces from the left) of the first line of text.

Tip: Negative values are allowed. The first line will be indented to the left if the value is negative.

Code & Output:

{text-indent: 350px;}

5. White-Space Property

Defines how white-space inside an element is handled in a text.

Syntax: {white-space: value;}

Values of White-Space Property:

There are 5 values of CSS white-space property:

  1. normal
  2. nowrap
  3. pre
  4. pre-line
  5. pre-wrap
