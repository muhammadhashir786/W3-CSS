Text-Shadow Property

The CSS text-shadow property adds shadow to the text.

We only specify the horizontal and vertical shadow.

Values of Text-Shadow Property

There are 3 values of text-shadow property:

  1. x-axis y-axis - define dimensions x-axis and y-axis
  2. color - define color of the shadow
  3. blur - define blur effect to the shadow

Syntax for Dimensions: {text-shadow: x-axis y-axis;}

In addition, you can add a shadow color and blur effect.

Syntax for all values: {text-shadow: x-axis y-axis blur color;}

Multiple Shadows

The text-shadow property also accepts multiple shadows for an element. Define each shadow in a comma-separated list.

Syntax: {text-shadow: x-axis y-axis blurr color, x-axis y-axis blurr color}

The following example shows a red and blue neon glow shadow:

Code & Output:

Text-shadow with red and blue neon glow:

{text-shadow: 0 0 6px #ff0000, 0 0 10px #0000ff;}
