CSS Selectors

CSS selectors are used to "find" (or select) the HTML elements that you want to style.

Types Of CSS Selectors

  1. Simple selectors (select HTML elements based on element name, id, class), universal and grouping selector.
  2. Combinator selectors (select elements based on a specific relationship between them)
  3. Pseudo-class selectors (select elements based on a certain state)
  4. Pseudo-elements selectors (select and style a part of an element)
  5. Attribute selectors (select elements based on an attribute or attribute value)

This page will explain the most basic CSS simple selectors only.

The CSS Element Selector

The element selector selects HTML elements based on the element name.

Explanation:

Here, all paragraph "p" elements on the below box will be "center-aligned" with a red text color:

Code:

p {
  text-align: center;
  color: red;
}

Output:

Every paragraph will be affected by the style.

Me too!

And me!


The CSS id Selector

The id selector uses the id attribute of an HTML element to select a specific element.

The "id" of an element will always be unique in all over the HTML document, so the "id" selector is used to select one unique element in the whole HTML document!

To select an element with a specific id, write a hash (#) character, followed by the id of the element.

Explanation:

The below CSS code will only be applied to the HTML element with id="para1":

Code:

#para1 {
  text-align: center;
  color: red;
}

Output:

This paragraph is not affected by the style. (id=para0)

Hello World! (id=para1)

not me! (id=para2)

Note: An id name cannot start with a number!

The CSS class Selector

The class selector selects HTML elements with a specific class attribute.

To select elements with a specific class, write a period (.) character, followed by the class name.

The value of "class" attribute can be same for multiple elements.

We can give many elements to same class

we can target many elements in the page using specific class.

Explanation:

In this example all HTML elements with class="center" will be red and center-aligned:

Code:

.center {
  text-align: center;
  color: red;
}

Output:

Red and center-aligned heading

Red and center-aligned paragraph.

You can also specify that only specific HTML elements should be affected or targetted by a class.

Explanation:

In this example only "p" HTML elements with class="center" will be blue and center-aligned:

Code:

p.center {
  text-align: center;
  color: red;
}

Output:

This heading will not be affected

This paragraph will be blue and center-aligned.

We can define multiple values to 1 single "class" attribute.

Explanation:

In this example the "p" element will be styled according to class="center1" and to class="large": 

p.center1 {
  text-align: center;
  color: red;
}

p.large {
  font-size: 300%;
}

Output:

This heading will not be affected

This paragraph will be red and center-aligned.

This paragraph will be red, center-aligned, and in a large font-size.

Note: A class name cannot start with a number!

The CSS Universal Selector

The universal selector (*) selects all HTML elements on the HTML document.

Explanation:

The below CSS code will affect or target every HTML element on the HTML document:

Code:

* {
  text-align: center;
  color: blue;
}

Output:

Hello world!

Every element on the page will be affected by the style.

Me too!

And me!

The CSS Grouping Selector

The grouping selector selects all the HTML elements with the same style definitions.

Explanation:

Look at the following CSS code (the h1, h2, and p elements have the same style definitions):

Code:

h1 {
  text-align: center;
  color: red;
}

h2 {
  text-align: center;
  color: red;
}

p {
  text-align: center;
  color: red;
}

Output:

Main Heading

Sub-Heading

This is some content in my webpage.

This line will not affected.

It will be better to group the selectors, to minimize the code.

To group selectors, separate each selector with a comma.

Explanation:

Look at the following CSS code (the h1, h2, and p elements have the same style definitions):

Code:

h1, h2, p {
  text-align: center;
  color: red;
}

Output:

Main Heading

Sub-Heading

This is some content in my webpage.

This line will not affected.
