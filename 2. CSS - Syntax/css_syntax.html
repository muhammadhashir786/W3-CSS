CSS Syntax

The CSS syntax consists into 2 parts:

  1. selector
  2. declaration block

The selector selects or points or targets to the HTML element that you want to style.

The declaration block contains one or more declarations separated by semicolons.

We define CSS property and value under the declaration block separated by a colon ";".

Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.

Explanation:

Here, all paragraph "p" elements on the below box will be "center-aligned" with a red text color:

Code:

p {
  color: red;
  text-align: center;
}

Output:

Every paragraph will be affected by the style.

Example Explained

  1. "p" is an element name selector in CSS (it points to the HTML element you want to style: "p" paragraph element).
  2. color is a property, and red is the property value
  3. text-align is a property, and center is the property value
