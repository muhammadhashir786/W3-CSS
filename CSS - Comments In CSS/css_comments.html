CSS Comments

Comments are used to explain the CSS code, and may help when you edit the source code at a later date.

Comments are also used to temporarily disable sections of CSS code within a stylesheet.

Syntax: /* comment goes here */

Code: /* this line has been commented */

Shortcut

The shortcut for comment or uncomment the whole single line is:

Press ctrl + /

If you want to comment multiple lines then select that lines and press the shortcut of comment or uncomment.
