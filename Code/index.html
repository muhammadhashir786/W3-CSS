1. Text Properties

There are 2 text properties in css:

  1. Color property
  2. Background-color property

a. The CSS Color Property

The CSS color property defines the text color of an HTML element

Code:

.firstPara {
    color: blue;
}

Output:
The color of this line is blue using css color property

b. The CSS Background-color Property

The CSS background-color property defines the background color of an HTML element

The background-color property is not the text property, it is used to change the background color of any HTML element. 

Code:

.secondPara {
    background-color: #2aff00;
}

Output:
The background-color of this line is green using css background-color property

2. Text Alignment

There are 5 text properties to align the text in css:

  1. text-align
  2. text-align-last
  3. vertical-align
  4. direction
  5. unicode-bidi

a. The CSS Text Align Property

The CSS text-align property defines the horizontal alignment of the text on the screen.

Values of Text-Align Property:

There are 4 values of text-align property:

  1. left - Aligns the text to the left
  2. right - Aligns the text to the right
  3. center - Centers the text
  4. justify - Stretches the lines so that each line has equal width

The following example shows left text alignment

Code:

.thirdPara {
  text-align: left;
}

Output:

this line is aligned text "left" in CSS which is also by default.

Note: left is default if text direction is left-to-right, and right is default if text direction is right-to-left:

The following example shows right text alignment

Code:

.fourthPara {
  text-align: right;
}

Output:

this line is aligned text "right" in CSS.

The following example shows center text alignment

Code:

.fifthPara {
  text-align: center;
}

Output:

this line is aligned text "center" in CSS.

The following example shows justify text alignment

Code:

.sixthPara {
  text-align: justify;
}

Output:

this line is aligned text "justify" in CSS.

b. The CSS Text-Align-Last Property:

The text-align-last property defines the text alignment of last line that displays in the webpage.

Values of Text-Align-Last Property:

There are 7 values of text-align-last property:

  1. auto - Default value. The last line is justified and aligned left
  2. left - The last line is aligned to the left
  3. right - The last line is aligned to the right
  4. center - The last line is center-alignedt
  5. justify - The last line is justified as the rest of the lines
  6. start - The last line is aligned at the start of the line
  7. end - The last line is aligned at the end of the line

The following example shows "auto" text alignment of the last line of paragraph: 

Code:

.seventhPara {
  text-align-last: auto;

c. The CSS Vertical-Align Property

The CSS vertical-align property defines the vertical alignment of an HTML element.

Values of Vertical-Align Property:

There are 9 values of vertical-align property:

  1. baseline - Default value. The element is aligned with the baseline of the parent
  2. length/% - Raises or lower an element by the specified length or percent
  3. sub - The element is aligned with the subscript baseline of the parent
  4. super - The element is aligned with the superscript baseline of the parent
  5. top - The element is aligned with the top of the tallest element on the line
  6. text-top - The element is aligned with the top of the parent element's font
  7. middle - The element is placed in the middle of the parent element
  8. bottom - The element is aligned with the lowest element on the line
  9. text-bottom - The element is aligned with the bottom of the parent element's font

3. Text Decoration

In the text decoration there is only 1 main property, which is called the "text-decoration" property that is used to decore the text

but Text Decoration means add lines in the text

a. The CSS Text-Decoration Property

The CSS text-decoration property defines the decorative lines to the text.

it is used to control the appearance of decorative lines on text.

Note:The main property is CSS text-decoration property.

we can define all the values of line decoration in CSS text-decoration property

but there is a keyword "line" that will add with "text-decoration" property, means "text-decoration-line" property

It is a shorthand property for the following individual properties:

we will learn how to use this shorthand property in CSS in the last of this section

  1. text-decoration-line - required
  2. text-decoration-color - optional
  3. text-decoration-style - optional
  4. text-decoration-thickness - optional

Note: The "text-decoration-line" property is required to define the type of line you need to decore text in the CSS "Text-Decoration" Property

a. The CSS Text-Decoration-Line Property

The CSS text-decoration-line property adds different types of decorative lines to the text.

Values of Text-Decoration-Line Property:

there are 4 values of text-decoration-line property:

  1. none - Default value. Displays no decoration line
  2. underline - The decoration line is displayed under the text
  3. overline - The decoration line is displayed over the text
  4. line-through - The decoration line is displayed through the text/from the center of text

Tip: You can combine multiple values, like overline and underline to display lines both over and under a text.

The following example shows values of text-decoration-line property:

Code:

.para25 {
  text-decoration-line: overline;
}

Output:

the value of "text-decoration-line" is set to "overline". 

Code:

.para26 {

}

Output:

the value of "text-decoration-line" is set to "line-through". 

Code:

  text-decoration-line: underline;
}

Output:

the value of "text-decoration-line" is set to "underline". 

Code:

.para28 {
  text-decoration-line: none;
}

Output:

the value of "text-decoration-line" is set to "none". 

Code:

.para29 {
  text-decoration-line: overline underline line-through;

the value of "text-decoration-line" is set to "overline underline line-through".
