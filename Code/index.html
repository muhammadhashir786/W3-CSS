CSS Text Chapter

1. Text Properties

There are 2 text properties in css:

  1. Color property
  2. Background-color property

a. The CSS Color Property

The CSS color property defines the text color of an HTML element

Code:

.firstPara {
    color: blue;
}

Output:
The color of this line is blue using css color property

b. The CSS Background-color Property

The CSS background-color property defines the background color of an HTML element 

Code:

.secondPara {
    background-color: #2aff00;
}

Output:
The background-color of this line is green using css background-color property

Look at CSS Color Values for a complete list of possible color values.

link: All About Color

Text Color of a Webpage

The default text color of a page is defined in the body selector in CSS. 

code:	

the below code defines only the text color of a webpage

body {
  color: orange;
}

2. Text Alignment

There are 5 text properties to align the text in css:

  1. text-align
  2. text-align-last
  3. vertical-align
  4. direction
  5. unicode-bidi

a. The CSS Text Align Property

The CSS text-align property defines the horizontal alignment of the text on the screen.

Values of Text-Align Property:

There are 4 values of text-align property:

  1. left - Aligns the text to the left
  2. right - Aligns the text to the right
  3. center - Centers the text
  4. justify - Stretches the lines so that each line has equal width

The following example shows left text alignment

Code:

.thirdPara {
  text-align: left;
}

Output:

this line is aligned text "left" in CSS which is also by default.

Note: left is default if text direction is left-to-right, and right is default if text direction is right-to-left:

The following example shows right text alignment

Code:

.fourthPara {
  text-align: right;
}

Output:

this line is aligned text "right" in CSS.

The following example shows center text alignment

Code:

.fifthPara {
  text-align: center;
}

Output:

this line is aligned text "center" in CSS.

The following example shows justify text alignment

Code:

.sixthPara {
  text-align: justify;
}

Output:

this line is aligned text "justify" in CSS.

Question: What is the defference between text alignment left and justify?

b. The CSS Text-Align-Last Property:

The text-align-last property defines the text alignment of last line that displays in the webpage.

Values of Text-Align-Last Property:

There are 7 values of text-align-last property:

  1. auto - Default value. The last line is justified and aligned left
  2. left - The last line is aligned to the left
  3. right - The last line is aligned to the right
  4. center - The last line is center-alignedt
  5. justify - The last line is justified as the rest of the lines
  6. start - The last line is aligned at the start of the line
  7. end - The last line is aligned at the end of the line

The following example shows "auto" text alignment of the last line of paragraph: 

Code:

.seventhPara {
  text-align-last: auto;
}
