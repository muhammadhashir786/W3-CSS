Text Decoration

Text Decoration is the main property in CSS to write all properties of text-decoration in shorhand way, see this in the last of this webpage

Properties & Values

  1. text-decoration-line (Required)
    1. none - Default value. Displays no decoration line
    2. overline - The decoration line is displayed over the text
    3. underline - The decoration line is displayed under the text
    4. line-through - The decoration line is displayed through the text/from the center of text

  2. text-decoration-color (Optional)
    1. give any color to the decorative line like {red, rgb(29, 173, 0), #0011ff}

  3. text-decoration-style (Optional)
    1. solid - Default value. Displays single line
    2. double - Displays double line
    3. dotted - Displays dotted line
    4. dashed - Displays dashed line
    5. wavy - Displays wavy line

  4. text-decoration-thickness (Optional)
    1. thickness value defines in the unit of length like (px, em or %).

Note: we can use multiple times "text-decoration-line" property.

The Shorthand Property

The formula of shorthand property is:

Syntax: {text-decoration: line color style thickness;}

Code: {text-decoration: overline rgb(255, 221, 0) wavy 8px;}
