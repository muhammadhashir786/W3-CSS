CSS Colors

In CSS, there is the most important property which is "color" that are specified by different types of values.

Values of Color Property:

  1. Color names - red
  2. RGB - rgb(255, 0, 0) for red color
    • RGBA - rgba(255, 0, 0, 0.5) for red color with 0.5 opacity
  3. HEX - #ff0000 for red color
  4. HSL - hsl(0, 100%, 50%) for red color
    • HSLA - hsla(0, 100%, 50%, 0.5) for red color with 0.5 opacity
